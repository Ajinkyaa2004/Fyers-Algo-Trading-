## ğŸ¯ User Flow Examples

### Example 1: Viewing Market Data
```
1. Navigate to "Live Market"
2. Enter symbol "NSE:SBIN-EQ"
3. Click "Search"
4. View quote card with LTP
5. Check market depth (bid/ask)
6. Scroll to view price history
7. Back to strategies for trading
```

### Example 2: Placing an Order
```
1. Navigate to "Strategies"
2. Scroll to "Live Trading" section
3. Enter symbol "NSE:SBIN-EQ"
4. Enter quantity "1"
5. Select "BUY" (or SELL)
6. Select "MARKET" (or LIMIT)
7. If LIMIT, enter price
8. Select "INTRADAY" (product type)
9. Keep "DAY" validity
10. Click "Place BUY Order"
11. See success notification
12. Order appears in Recent Orders
```

### Example 3: Checking Portfolio
```
1. Navigate to "Portfolio"
2. View Holdings table
   - Symbols held
   - Current prices
   - P&L percentages
3. Check Open Positions
4. Review Order History
5. View Margins & Funds
6. Use Refresh button if needed
```
